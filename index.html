OBS Setup Steps
  1. Open OBS and go to your scene
  2. Click + in Sources → select Browser
  3. Give it a name (e.g. Live Captions) and click OK

TikTok Studio Setup

TikTok Studio doesn't support browser sources, so CaptionCast spawns a floating caption window that sits on top of everything on your screen. You capture it in TikTok Studio as a Window Capture source.

💡 Tip: Set the width to match your stream canvas (usually 1920). Height of 160–220px works well for most font sizes. Changes apply when you relaunch the window.
TikTok Studio Steps
  1. Click Launch Floating Window above — a transparent caption window will appear at the bottom of your screen
  2. Open TikTok Studio and go to your scene
  3. Click + to add a source → select Window Capture
  4. Choose CaptionCast Overlay from the window list
  5. Resize and position the source at the bottom of your scene
  6. Press Start Captions in the Stream tab — done!
🪟 The floating window is draggable — click and drag it anywhere on screen. It always stays on top so it won't get hidden behind TikTok Studio. Your caption style settings from the Style tab apply here too.
What is Deepgram?

Deepgram is the speech-to-text engine powering CaptionCast. It's free to start — $200 of credits included, which is hundreds of hours of live captioning at no cost.

Getting Your Free API Key
  1. Go to console.deepgram.com/signup — create a free account (no card needed)
  2. Verify your email address
  3. Log in → go to API Keys in the left sidebar
  4. Click Create a New API Key
  5. Give it any name (e.g. CaptionCast) → click Create Key
  6. Copy the key immediately — it's only shown once! Paste it in the Stream tab.
🔐 Your key stays on your computer. It's stored locally and only ever sent directly to Deepgram — never shared.
How It Works
  1. CaptionCast captures audio from your selected microphone
  2. Audio streams in real-time to Deepgram Nova-2 for transcription
  3. Text is relayed to your OBS overlay via a local connection
  4. Captions auto-clear after the silence delay you set
  5. Adjust font, size, colour and outline in the Style tab anytime
